# Password Generator

My first Python project: a simple random password generator.

## Features
- Generates passwords of any length
- Uses uppercase letters, lowercase letters, numbers, and symbols
- Validates user input

## How to Run

1. Install Python
2. Clone this repository
3. Run:

python password-generator.py

## What I Learned
- Importing and using modules
- Working with strings
- Using while loops for input validation
- Using for loops for repetition
- Generating random values
